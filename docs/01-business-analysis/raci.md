# RACI Matrix

> **Last Updated:** 2026-03-09

## Purpose

Defines **R**esponsible, **A**ccountable, **C**onsulted, and **I**nformed roles for key project activities.

## Matrix

| Activity | Product Owner | Tech Lead | Dev Team | QA | Stakeholder |
|----------|:---:|:---:|:---:|:---:|:---:|
| Requirements gathering | A | C | I | I | R |
| Architecture decisions | C | R/A | C | I | I |
| Sprint planning | R | A | R | C | I |
| Code review | I | R | R/A | I | — |
| Release sign-off | A | C | I | R | I |
| Incident response | I | R | R | I | I |

> **Legend:** R = Responsible, A = Accountable, C = Consulted, I = Informed
