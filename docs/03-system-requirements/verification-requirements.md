# Verification Requirements

> **Standard:** ISO/IEC/IEEE 29148:2018 §6.7  
> **Last Updated:** 2026-03-09

## Purpose

Specifies HOW each requirement shall be verified (inspection, analysis, demonstration, or test).

## Verification Methods

| Method | Description | When Used |
|--------|-------------|-----------|
| **Inspection** | Review of documents or code | Standards compliance |
| **Analysis** | Mathematical or logical reasoning | Performance models |
| **Demonstration** | Showing capability through operation | UI workflows |
| **Test** | Executing against defined conditions | Functional/NFR validation |
