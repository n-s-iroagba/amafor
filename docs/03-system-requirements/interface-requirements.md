# Interface Requirements

> **Standard:** ISO/IEC/IEEE 29148:2018 §6.6.3  
> **Last Updated:** 2026-03-09

## Purpose

Specifies external interfaces the system must support — APIs, third-party integrations, hardware, and user interfaces.

## External Interfaces

| ID | Interface | Type | Direction | Protocol | Status |
|----|-----------|------|-----------|----------|--------|
| IR-001 | REST API (Client ↔ Server) | Software | Bidirectional | HTTPS/JSON | Active |
| IR-002 | Paystack Payment Gateway | Software | Outbound | HTTPS | Active |
| IR-003 | SMTP Email Service | Software | Outbound | SMTP/TLS | Active |
